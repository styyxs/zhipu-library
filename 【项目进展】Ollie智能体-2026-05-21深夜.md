## 一、核心阻塞问题

### 问题：Cloudflare Worker 域名被墙

**现象（森哥反馈）：**
- 用户使用时报"一直卡在播放动画那里"
- TTS语音不播放，AI图片不生成

**截图标明的根因：**
- 错误代码：`ERR_CONNECTION_TIMED_OUT`
- 问题域名：`ollie-mimi-api.962324377.workers.dev`
- 结论：**`.workers.dev`域名在国内被防火墙屏蔽**，无法访问

**影响的两个Worker：**

| Worker | 用途 | 域名（被墙） |
|--------|------|-------------|
| ollie-mimi-api | MiniMax TTS语音合成 | ollie-mimi-api.962324377.workers.dev |
| ollie-zhipu-library | 智谱图片生成API | ollie-zhipu-library.962324377.workers.dev |

**项目其他部分状态：**

## 二、解决方案（已确认，待执行）

### 方案：给Worker绑定自定义域名

**原理：**
- `.workers.dev` 是Cloudflare默认子域名，被重点封锁
- `api.ollie.zhipu-library.top` / `image.ollie.zhipu-library.top` 是自定义域名，走正常CDN，不被屏蔽

**森哥在Cloudflare Dashboard操作：**

1. 进入 [Cloudflare Dashboard](https://dash.cloudflare.com)
2. 选择 **ollie-mimi-api** Worker → **"Domains"** 标签 → **"Add route"**
3. 添加子域名：`api.ollie.zhipu-library.top`
4. 对 **ollie-zhipu-library** Worker 同样操作，添加：`image.ollie.zhipu-library.top`

**验证：**
- 绑定后国内网络直接 curl 测试（不用VPN）：
  ```
  curl -sL https://api.ollie.zhipu-library.top/
  curl -sL https://image.ollie.zhipu-library.top/
  ```

**代码改动（域名绑定好后小爱执行）：**

修改 `js/tts.js`：
```javascript
// 改前（被墙）
const API_BASE = 'https://ollie-mimi-api.962324377.workers.dev';

// 改后（自定义域名）
const API_BASE = 'https://api.ollie.zhipu-library.top';
```

修改图片API调用代码（js/app.js 或相关文件）：
```javascript
// 改前（被墙）
const IMAGE_API = 'https://ollie-zhipu-library.962324377.workers.dev';

// 改后（自定义域名）
const IMAGE_API = 'https://image.ollie.zhipu-library.top';
```

然后 GitHub push → Cloudflare Pages 自动部署 → 验证

## 五、经验总结

- **.workers.dev 域名问题**：Cloudflare默认`.workers.dev`子域名在国内被严格封锁，所有Worker服务都要用自定义域名
- **教训**：以后部署Worker，第一时间绑定自定义域名，不要用默认的`.workers.dev`
- **排查方法**：截图错误代码 → 直接定位根因，比猜测快得多

## 六、关于国内服务器的思考

**今天卡点分析：**

| 问题 | 根因 | 国内服务器能解决吗 |
|------|------|-----------------|
| `.workers.dev`被墙 | 海外服务器被封 | ✅ 能 |
| GitHub有时慢/连不上 | 海外服务器问题 | ✅ 能 |
| 域名访问速度 | 跨境延迟 | ✅ 会更快 |

**但国内服务器有致命问题：备案**
- 国内大陆服务器必须域名备案
- 备案周期：2-4周
- 备案材料：需要企业营业执照（智联招聘河南分公司公户）
- `.zhipu-library.top`能不能备，要看森哥有没有企业资质

**结论：短期最优解是继续用Cloudflare，绑定自定义域名解决被墙问题**

**长期方案（如果项目要持续运营）：**
- 买腾讯云/阿里云轻量应用服务器（2核4G约100-200/月）
- TTS和图片API换成国内服务商（阿里云语音合成、腾讯云混元图片）
- 但要先确认域名能备案
